1 Data Sources
This is where all your data lives originally ‚Äî your apps, websites, sensors, and files. Think of these as the "raw ingredients" before cooking.
Transactional Databases (OLTP)
Databases that store day-to-day business data ‚Äî every order, user signup, or payment.
MySQL PostgreSQL SQL Server Oracle MongoDB
Like a cash register that records every sale as it happens.
Enterprise Applications
Big software systems that large companies use to run their entire business.
SAP Oracle ERP Salesforce
SAP ‚Äî Manages everything: inventory, HR, finance, manufacturing
Oracle ERP ‚Äî Similar to SAP, handles company operations
Salesforce ‚Äî Tracks customers, sales, and deals
Like the brain of a company ‚Äî knows about employees, products, customers, and money.
SaaS Applications
Online tools that companies subscribe to for specific tasks.
HubSpot Stripe Shopify Zendesk
HubSpot ‚Äî Marketing emails, lead tracking
Stripe ‚Äî Online payments
Shopify ‚Äî Online store platform
Zendesk ‚Äî Customer support tickets
Files & Logs
Plain files sitting on computers or shared drives.
CSV Excel JSON XML Server Logs
CSV/Excel ‚Äî Spreadsheet data
JSON/XML ‚Äî Structured text files used by apps
Server Logs ‚Äî Records of what happened on a website or app
APIs
Ways for apps to share data with each other automatically.
REST APIs Webhooks GraphQL
Like a waiter taking orders between the kitchen (data) and customers (apps).
Events & Clickstream
Tracks what users do on websites and apps ‚Äî every click, scroll, and button press.
Segment Mixpanel Google Analytics
Like security cameras recording every movement of shoppers in a store.
IoT & Sensors
Data from physical devices ‚Äî machines, vehicles, wearables.
Machine Sensors GPS Data Smart Devices
Like a fitness watch tracking your heartbeat and steps.

2 Data Ingestion
Moving data from sources into a central place. Like collecting ingredients from different shops and bringing them to your kitchen.
Batch Ingestion (EL in ELT)
Copies data in chunks at scheduled times ‚Äî every hour, daily, etc.
Fivetran Airbyte Stitch Hevo
Fivetran ‚Äî Paid, very easy, 300+ connectors ready to use
Airbyte ‚Äî Free/open source alternative to Fivetran
Stitch ‚Äî Simpler, good for small teams
Hevo ‚Äî Popular in India, no coding needed
Like a delivery truck that picks up goods from suppliers every morning.
Streaming Ingestion
Moves data instantly as it happens ‚Äî no waiting.
Apache Kafka Confluent AWS Kinesis Azure Event Hubs
Kafka ‚Äî Open source, industry standard for streaming
Confluent ‚Äî Kafka but managed (easier to use)
Kinesis ‚Äî AWS's streaming service
Event Hubs ‚Äî Azure's streaming service
Like a live TV broadcast ‚Äî data flows continuously in real-time.
Change Data Capture (CDC)
Only captures what changed ‚Äî new, updated, or deleted records. Efficient for large databases.
Debezium Fivetran CDC AWS DMS
Like tracking only the edits made to a document, not copying the whole thing every time.

3 Data Storage
Where you keep all your collected data. Like a warehouse or refrigerator where ingredients are stored before cooking.
Cloud Object Storage (Data Lakes)
Cheap storage for massive amounts of raw data ‚Äî files, images, logs, anything.
AWS S3 Google Cloud Storage Azure Blob Storage
S3 ‚Äî Amazon's storage, most popular
GCS ‚Äî Google's equivalent
Blob ‚Äî Microsoft Azure's storage
Like a massive storage unit ‚Äî dump everything here, sort it later.
File Formats
How data is packaged and stored. Some formats are faster to read, some save more space.
Parquet Avro ORC Delta Iceberg
Parquet ‚Äî Most popular for analytics, compressed, fast
Avro ‚Äî Good for streaming data
Delta/Iceberg ‚Äî Modern formats that add database-like features to files
Like choosing between a ZIP file or a folder ‚Äî same content, different packaging.
Data Warehouses
Organized storage built for fast analysis. Data is cleaned and structured here.
AWS ‚Üí Redshift GCP ‚Üí BigQuery Azure ‚Üí Synapse Snowflake Databricks
Redshift ‚Äî Amazon's warehouse
BigQuery ‚Äî Google's warehouse (you've used this!)
Synapse ‚Äî Microsoft's warehouse
Snowflake ‚Äî Works on any cloud, very popular
Databricks ‚Äî Lakehouse (warehouse + lake combined)
Like an organized library ‚Äî books are sorted and easy to find quickly.

4 Data Transformation
Cleaning and reshaping raw data into useful formats. Like cooking raw ingredients into a proper meal.
ELT (Modern - in-warehouse)
Transform data inside the warehouse using SQL. The modern, preferred approach.
dbt Dataform (BigQuery) Coalesce (Snowflake) SQLMesh
dbt ‚Äî Industry standard, works everywhere, free
Dataform ‚Äî Google's tool, built into BigQuery
Coalesce ‚Äî Visual tool for Snowflake
SQLMesh ‚Äî Newer alternative to dbt
Like cooking inside the restaurant kitchen (warehouse) itself.
ETL (Traditional - outside warehouse)
Transform data before loading into warehouse. Older approach, still used in big companies.
Informatica Talend SSIS DataStage Pentaho
Informatica ‚Äî Enterprise giant, expensive but powerful
Talend ‚Äî Has free open-source version
SSIS ‚Äî Microsoft's tool for SQL Server
DataStage ‚Äî IBM's enterprise tool
Like preparing food at a central kitchen, then delivering to restaurants.
Streaming Transformation
Transform data as it flows in real-time ‚Äî no waiting for batches.
Apache Flink Spark Streaming ksqlDB
Flink ‚Äî Powerful real-time processing
Spark Streaming ‚Äî Part of Spark, handles batch + stream
ksqlDB ‚Äî SQL on top of Kafka streams
Like a sushi chef preparing food as customers order ‚Äî instant.

Tools that let you run SQL queries directly on data lakes (files) without loading into a warehouse first. Useful for quick exploration.

Presto/Trino ‚Äî Open source, query any data source
Athena ‚Äî AWS service, query S3 files with SQL
Dremio ‚Äî Fast queries on data lakes

Scheduling and managing workflows ‚Äî making sure tasks run in the right order at the right time. The "project manager" of data pipelines.

Airflow ‚Äî Industry standard, open source, most popular
Prefect ‚Äî Modern alternative, easier to use

Tools that turn data into charts, dashboards, and reports. This is what business users actually see and use.

Powerful tools used by large companies. Usually expensive.

10 DevOps & Infrastructure
Tools that help deploy, manage, and version your code and infrastructure. The foundation that keeps everything running.
Containerization
Package your code with everything it needs to run anywhere.
Docker Kubernetes
Docker ‚Äî Creates containers (portable app packages)
Kubernetes ‚Äî Manages many containers at scale
Like shipping containers ‚Äî your stuff stays safe and works the same everywhere.
Version Control
Track changes to code. Collaborate without overwriting each other's work.
Git GitHub GitLab
Git ‚Äî The system that tracks changes
GitHub ‚Äî Website to store and share Git code
GitLab ‚Äî Alternative to GitHub with more built-in tools
Like "Track Changes" in Word, but for code ‚Äî see who changed what and when.
Infrastructure as Code
Define your servers, databases, and cloud resources in code files instead of clicking buttons.
Terraform Pulumi
Terraform ‚Äî Most popular, works with all clouds
Pulumi ‚Äî Use real programming languages
Like a blueprint for a building ‚Äî anyone can recreate the same setup from the plan.

Example: Global Manufacturing Company

A manufacturing giant with 10,000 employees runs SAP for inventory and finance, Salesforce for sales, and collects IoT sensor data from factory machines. Debezium captures every database change in real-time. Confluent Kafka streams millions of events per second. Raw data lands in S3 as Delta Lake tables. Databricks processes both batch and streaming data. Legacy Informatica jobs still run alongside modern dbt models. DataHub catalogs 5,000+ tables so analysts can find what they need. Finance uses Power BI, sales uses Tableau, and executives use Looker dashboards.
